# 有效的字母异位词 LC 242
给定两个字符串 s 和 t ，编写一个函数来判断 t 是否是 s 的字母异位词。
当key值确定时，首先考虑以**数组**替代hashmap。
```java
class Solution {
    public boolean isAnagram(String s, String t) {
        int[] temp = new int[26];
        for(int i = 0;i < s.length();i++){
            temp[s.charAt(i) - 97]++;
        }

        for(int i = 0;i < t.length();i++){
            temp[t.charAt(i) - 97]--;
            if( temp[t.charAt(i) - 97] < 0){
                return false;
            }
        }

        for(int i = 0;i < 26;i++){
            if(temp[i] > 0){
                return false;
            }
        }

        return true;
    }
}
```
